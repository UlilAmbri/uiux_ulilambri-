## _Kelebihan Online Usability Testing_
#####_Kelebihan_
```
1. Dapat menemukan user dengan mudah dan cepat
2. Lebih cepat dan lebih murah
3. Jangkauan user yang sangat luas
4. Efesien
```

#####_Discovery Usability Testing_
Sebuah metode untuk menggali insight dari user terhadap 1 tampilan.

#####_General Tips_
```
1. Buat Misi sesimple mungkin
2. Ikuti user flow yang sudah dirancang sebelumnya
3. pastikan prototype anda berjalan dengan lancar sesuai dengan rancangan anda inginkan
4. Hindari bahasa teknis
5. Tentukan Parameter hasil dari testing anda
```
